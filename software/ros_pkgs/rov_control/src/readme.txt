# T1 Controller Layout

Left Joystick 
    - x = axis 0, y = axis 1
    - (l=-1,r=1), (u=-1,d=1)
    - y axis up and down will be remap to 1 and -1 for consistency

Right Joystick
    - Sort of Broken

Dpads
    - (x,y) = hat 0    
    - (l=-1,r=1), (u=1,d=-1)
    - Note: y axis direction default configuration is different from the joystick default

Buttons
    - a = button 0
    - b = button 1
    - x = button 2
    - y = button 3
